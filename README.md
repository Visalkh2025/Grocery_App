# Grocery App - Build Flutter UI

This repository contains the frontend implementation of a grocery shopping application built with Flutter.

## About The Project

This project is a Flutter-based mobile application that provides a user interface for a grocery shopping app. It is designed to showcase the UI/UX of a modern grocery application.

**Please note:** This is a frontend-only implementation. All the data used in the application is mocked and there is no backend server connected to this application.

### Features

*   User authentication (mocked)
*   Product browsing and search
*   Product details view
*   Shopping cart functionality
*   User profile management

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

*   Flutter SDK: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)
*   Dart SDK: [https://dart.dev/get-dart](https://dart.dev/get-dart)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/Project-Name.git
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
